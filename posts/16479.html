hexo | 支持资源压缩部署加速网站访问
  1. 使用方法

使用方法

你的网站文件夹，举例为website。

  1. 命令行进入到webiste目录。

  2. 执行命令安装hexo插件：
    npm install hexo-all-minifier

  3. 添加如下配置到_config.yml文件（注意不是A4主题的配置文件）

    #资源进行打包压缩
all_minifier: true

  4. 执行命令查看效果

    hexo clean && hexo g && hexo s
